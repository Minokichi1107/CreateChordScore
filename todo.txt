# Task: Add Export / Import for Custom Chord Diagrams

## Context

This is a chord chart editor web app.

Project structure:

* index.html
* style.css
* app.js

The application already has:

1. Song project export/import (JSON)
2. Custom chord diagrams stored in localStorage

The chord diagrams stored in localStorage act as the user's personal chord library.

---

# Problem

Custom chord diagrams are stored only in localStorage.
If the browser storage is cleared or the user changes devices, these diagrams are lost.

---

# Goal

Add the ability to:

• Export custom chord diagrams to a JSON file
• Import that JSON file to restore the diagrams

---

# Requirements

1. Read the existing custom chord diagram data from localStorage.

2. Create an export function that downloads the data as a JSON file.

3. Create an import function that reads a JSON file and restores the data back into localStorage.

4. The JSON structure should match the **existing internal data structure** to avoid compatibility issues.

---

# Constraints

Important:

• Do NOT change the existing localStorage structure
• Do NOT redesign existing data models
• Do NOT refactor unrelated code
• Only add backup/restore functionality

---

# Expected Result

User can:

Export custom chord diagrams → JSON file
Import JSON file → restore diagrams into localStorage

This should work independently from the existing song project JSON system.
