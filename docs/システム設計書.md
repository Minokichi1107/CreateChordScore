# CreateChordScore

## システム設計書 (System Design Document)

---

# 1. システム概要

## 1.1 ツール目的

CreateChordScore は **音声ファイルを解析しコード譜を生成・編集するツール**である。

主用途

* 音声ファイルからコード進行を取得
* コード譜の編集
* ギターコードダイアグラム表示
* プロジェクト保存 / 読み込み

対象ユーザー

* ギター演奏者
* コード譜作成者
* 音源からコード進行を抽出したいユーザー

---

# 4. 状態管理（State）

状態は主に **app.js に集中管理される。**

---

# 4.1 Project状態

```javascript
project = {

  title
  artist
  beats
  audioFile
  lines[]
  palette[]

}
```

| フィールド     | 内容     |
| --------- | ------ |
| title     | 曲名     |
| artist    | アーティスト |
| beats     | 拍子     |
| audioFile | 音声     |
| lines     | コード譜行  |
| palette   | 使用コード  |

---

# 4.2 UI状態

```javascript
uiState = {

  focLine
  tapIdx
  diagOn
  rbHits

}
```

| 状態      | 内容           |
| ------- | ------------ |
| focLine | フォーカス行       |
| tapIdx  | TAPモードインデックス |
| diagOn  | ダイアグラム表示     |
| rbHits  | 置換回数         |
